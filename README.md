# Url Shortener

This repo is for me to learn what goes into shortening a url. I will be using the following technologies:
 - React
 - Python
 - Firebase
 - Bootstrap

This project is mostly for work.

## How to run
1. Clone the repo
2. Run `npm install` in the root directory
3. Run `npm start` in the root directory
4. Go to `localhost:3000` in your browser
